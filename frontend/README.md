# hey-lets-meet — Frontend

React + TypeScript + Vite frontend for the hey-lets-meet app.

## Requirements

- Node.js v20+
- npm

## Install

```sh
npm install
```

## Run

```sh
npm run dev
```

Opens at `http://localhost:5173`. Proxies `/api` and `/auth` requests to the backend at `http://localhost:8080`.

## Build

```sh
npm run build
```

## Test

```sh
npm test
```

## Lint

```sh
npm run lint
```

## Format

```sh
npm run format        # fix files in place
npm run format:check  # check only, no changes
```
